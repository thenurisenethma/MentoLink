# MentoLink 📚🎓

MentoLink is a mobile tutoring marketplace application designed to connect school students with university student mentors for academic support, tutoring services, and learning guidance.

The platform aims to provide affordable, accessible, and flexible learning opportunities while creating mentoring and income opportunities for university students.

---

## Project Overview

Many students face challenges when searching for reliable tutors due to:

- Difficulty finding trustworthy tutors
- High tutoring costs
- Limited scheduling flexibility
- Poor communication channels
- Inconsistent tutoring quality

MentoLink addresses these issues through a centralized mobile platform where students can discover, connect with, and learn from tutors efficiently.

---

## Features

### Student Features
- User registration and login
- Search tutors by subject and availability
- View tutor profiles
- View ratings and reviews
- Book tutoring sessions
- Receive notifications and reminders
- Access learning materials and recordings
- Secure online payments

### Tutor Features
- Tutor registration and profile management
- Set availability schedules
- Upload learning resources
- Receive booking requests
- Manage tutoring sessions

### Admin Features
- User management
- Tutor verification
- Monitor platform activities

## System Requirements

### Functional Requirements
- User authentication
- Tutor search and matching
- Session booking system
- Secure payment system
- Notification management
- Learning resource management

### Non-functional Requirements
- Fast response time (<3 seconds)
- Secure user authentication
- High availability
- Scalable database structure
- User-friendly interface
